位运算不再神秘

计算机中的乘法就是用到了位运算的知识！

例如15*13
就会把13拆解成8+4+1 的形式 也就是 2的0次方 2的2次方和 2的三次方
分别对应了 15<<0 (15左移0位） + 15<<2 (15左移两位) + 15 << 3 (15左移三位)的和
这样对于计算机来说，处理起来绝对相当简单！

计算的减法其实就是加法

使用补码的方式进行运算，具体可以见计算机组成原理，设计的非常巧妙

计算机中除法非常消耗时间

除法其实就是所有位数左移的过程，但是浮点数的除法，小数的处理真的非常耗时间。

基本的位运算符号

含义 运算符 例子
左移 << 0011 => 0110
右移 >> 0110 => 0011
按位或 一个竖线 0011 竖线 1011 => 1011
按位与 & 0011 & 1011 => 0011
按位取反 ~ 0011 => 1100
按位异或（相同0不同1） ^ 0011 ^ 1011 => 1000
